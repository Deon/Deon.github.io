EventWall

MongoDB, Express, AngularJS, Node.js

EventWall is an event hub that is being built for University of Waterloo students. Anyone can see and post events to the site to discover activites, clubs, and teams. EventWall is also hooked into the UW API to query for CECA events, as well as the FEDS and EngSoc calendars. Students can login using local authentication, Facebook, Twitter, etc. using OAuth with Passport.

Trying to figure out how to split the cost of a receipt or bill? Sounds easy enough. What if the cost isn't being split evenly? That's another story. That's why there's Splitter.

LinAlg Toolkit

AngularJS, Flask

The LinAlg Toolkit reduces matrices and provides their properties. It also does complex number operations as well as conversion between polar and rectangular form.

Complex number calculations are done on the front-end using JS. Matrix operations are handled by SymPy and then rendered using MathJax.
